# Shop Data Analysis Project

This project is designed for storing, cleaning, and analyzing a store’s sales and user data using Python, pandas, SQLite, matplotlib, seaborn, and datetime.

## Features
- Read CSV files
- Store data in SQLite
- Run SQL queries
- Analyze sales data
- Create visual charts

## Requirements
- Python 3.10+
- pandas
- sqlite3
- matplotlib
- seaborn
- datetime
- logging

## Project Structure
- `database_manager.py`
- `orders.csv`
- `analytics.py`
- `cleaned_orders.csv`
- `users.csv`
- `shop_data.db`
- `cleaning_order_data.py`

## Outputs
- SQLite database file: `shop_data.db`
- Log file: `data_processing.log`
- Analytical charts and plots
